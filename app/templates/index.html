API Documentation

Endpoints

Unix

  • GET /unix - Retrieve data
  • GET /unix?limit=3000 - Default limit is 10000

Underlying Candles

  • GET /underlying - Retrieve data (Autosorted by most recent time)
  • GET /underlying?limit=3000 - Default limit is 10000
  • GET /underlying?timeframe=FiveSecs - Get underlying candles by timeframe

Option Candles

  • GET /option - Retrieve data (Autosorted by most recent time)
  • GET /option?limit=3000 - Default limit is 10000
  • GET /option?tags=Call,5min - List tags separated by comma for filtering

Option Tags

  • GET /option/tags - Get list of current option tags

Option Candle Performance

  • GET /option/performance - Retrieve data (Autosorted by most recent time)
    • Use same tags as with options, will just have attached performance data

Example Request

curl -X GET http://localhost:5000/api/data
